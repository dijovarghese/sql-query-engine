Project Name: SQL Query Engine
Description: The project is to develop a SQL query processing engine which can process and generate output for a limited set of SQL select statements from a given CSV file.

Language: Java 1.8
IDE: Eclipse Mars.2 Release (4.5.2)

Program Inputs
Input File Type: CSV (sample file attached "sql_engine_dataset.csv")
Exit Command: EXIT (use this exit command to exit from the program)
